# 公众号源码排版助手

源码公开的微信公众号后台排版与源码增强工具，由熵灵硅界维护。

## 功能

- HTML 源码模式：点击 `HTML` 进入源码，再次点击 `HTML` 原样写回并退出。
- 图片参数工具：点击文章图片，在图片侧边调圆角、尺寸、间距、阴影、发光、色彩、相框、图注、圆形等。
- SVG 动效工具：支持选择 2 到 9 张图片生成淡入、擦除、滑入、轮播、叠入等 SVG 动效。
- SVG 成品管理：已生成的 SVG 动效块可继续调整、删除或还原为静态图。

## 设计原则

- 不自动清理源码。
- 不自动备份。
- 不偷偷优化或修复用户代码。
- 只有用户点击按钮或拖动参数时，才改对应内容。

## 安装

1. 打开 `chrome://extensions/`。
2. 开启开发者模式。
3. 加载已解压的扩展程序。
4. 选择 `wechat-mp-source-layout` 文件夹。
5. 刷新微信公众号图文编辑页。
