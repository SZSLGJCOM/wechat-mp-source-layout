# 更新日志

这里记录公众号源码排版助手的正式发布。

## v0.12 · 2026-07-23

### 通用图片来源

- 原图读取不再限制微信图片域名，正文中的任意 HTTPS 光栅图片统一进入同一套读取、烘焙、微信上传与事务写回流程。
- 按真实文件签名识别图片类型；原图解码失败时自动降级，WebP、AVIF、GIF、BMP 在进入 SVG 烘焙前统一标准化为 PNG，避免二次解码失败。
- 原图在调参与上传期间始终留在原位；微信粘贴图只作为临时上传载体，获取 CDN 地址后由同一快照事务清理并仅更新原图来源，保留尺寸、裁剪、样式、效果参数和连续选中状态。
- 微信原生粘贴产生的正文事件纳入同一上传事务，不再误报正文冲突；连续调节只更新原图，所有临时载体一次清理。
- 保留图片类型、16MB 上限、HTTPS、可信编辑器来源与上传后微信 CDN 结果校验，读取失败继续恢复已提交状态。

### 极简 HTML 源码模式

- 点击工具栏 HTML 进入纵向格式化源码，再次点击自动保存并返回正文。
- 删除源码区标题、重新读取、格式化、复制、保存、保存并退出、关闭及底部状态栏，只保留行号与代码区域。
- 保存与文章同步继续使用会话锁和内容冲突校验，避免切换文章时写回错误正文。

## v0.11 · 2026-07-23

### 微信图片来源与编辑事务

- 像素效果统一改走当前正文编辑器的文件粘贴链：由微信生成 CDN 资产和原生图片属性，确认后撤回粘贴占位，再与全部样式一次写回。
- 移除私有上传接口及 ticket、scene 等内部参数依赖；编辑器不接收合成粘贴时保留原图并明确失败。
- 修复透明度等样式误删图片来源、横竖图比例写反、烘焙与普通样式竞态，以及候选图片迟到、原位替换和清理失败导致的图片消失问题。
- 图片任务改用稳定身份、修订号和统一提交门，文章切换、节点重建及连续写入失败时仍保留明确的回滚与清理所有权。

### HTML 源码模式

- 保留原始 HTML，格式化改为显式操作；新增重新载入、复制、保存、保存并退出和关闭的完整操作流。
- 保存、重载、文章同步与关闭采用统一会话锁，避免旧文章写回、永久只读和未保存内容误丢失。
- 优化高亮、行号、手机预览与文章切换监听，减少重复解析和非必要 DOM 工作。

## v0.10 · 2026-07-23

### 首个正式版本

- 提供 HTML 源码编辑、真实手机阅读预览、图片处理、多图 SVG 动效生成与 SVG 成品管理。
- 图片选中、拖动与缩放使用微信编辑器原生能力，扩展负责精确参数、视觉效果与可靠恢复。
- 描边、色彩、阴影、发光与羽化按照图片 Alpha 轮廓烘焙，并作为本地图片上传到编辑器。
